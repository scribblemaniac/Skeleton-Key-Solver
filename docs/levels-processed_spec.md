# Specifications for levels-processed.dat

Legend for symbols found in levels-processed.dat:

```
a = space
* = obstacle
. = dead space (impossible to solve if key enters this)
k = key
x = chest
s = switch
l = door closed
O = door opened
```

Additional characters used during Solver processing:

```
X = filled chest
# = key in door
S = key on switch
```
